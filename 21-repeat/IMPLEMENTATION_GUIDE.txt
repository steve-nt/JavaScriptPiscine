Concept 1: What are Strings?
-----------------------------
A string is just a piece of text. In JavaScript, we put text in quotes:

  "Hello"          // This is a string
  "repeat me!"     // This is also a string
  ""               // This is an EMPTY string (no text)

Strings are like sentences you can manipulate. Just like you can say "Hi" three
times to get "HiHiHi", we can repeat strings in code!

Concept 2: What are Parameters/Arguments?
------------------------------------------
When you call a function, you pass information INTO it using parameters:

  repeat("Hi", 3)
         ‚Üë    ‚Üë
      Parameter 1: The string to repeat
      Parameter 2: How many times

Think of it like ordering food:
  order("pizza", 2)  ‚Üí  You're ordering 2 pizzas
  repeat("*", 5)    ‚Üí  You're repeating "*" 5 times

Concept 3: The Return Value
----------------------------
When a function is done, it gives you something back. We call this the "return
value":

  function add(a, b) {
    return a + b;  // This is the return value
  }

  let result = add(2, 3);  // result is now 5

For our repeat function:
  let repeated = repeat("Hi", 3);  // repeated will be "HiHiHi"

Concept 4: Loops - Doing Something Multiple Times
--------------------------------------------------
A loop is a way to repeat code automatically. Imagine you want to copy "Hi"
three times:

Without a loop (boring):
  let result = "Hi" + "Hi" + "Hi";

With a loop (smart!):
  let result = "";
  for (let i = 0; i < 3; i++) {
    result = result + "Hi";
  }

A FOR LOOP works like this:
  for (let i = 0; i < 3; i++)
  ‚îî‚îÄ Create a counter starting at 0
  ‚îî‚îÄ Keep going while i is less than 3
  ‚îî‚îÄ After each loop, add 1 to i

So it runs 3 times: when i=0, i=1, and i=2. Perfect for repeating things!

Concept 5: Recursion - Functions Calling Themselves
----------------------------------------------------
Recursion is when a function calls ITSELF. It's like looking in a mirror that
reflects another mirror‚Äîit goes on and on!

Example:
  function countdown(n) {
    if (n <= 0) {
      return "Blastoff! üöÄ";  // BASE CASE - stop here!
    }
    return n + " " + countdown(n - 1);  // Call myself with a smaller number
  }

  countdown(3)  ‚Üí  "3 2 1 Blastoff! üöÄ"

For our repeat function, recursion would work like:
  repeat("Hi", 3) = "Hi" + repeat("Hi", 2)
                          = "Hi" + "Hi" + repeat("Hi", 1)
                                        = "Hi" + repeat("Hi", 0)
                                                 = ""  (BASE CASE)

**IMPORTANT**: Recursion needs a "base case" (when to stop) or it loops forever!

Concept 6: String Concatenation
--------------------------------
Concatenation means "joining strings together":

  "Hello" + " " + "World"  ‚Üí  "Hello World"

In our repeat function, we'll use concatenation to build up the final string by
adding the input string multiple times.

        ‚Üí Return string + repeat(string, n - 1)

Step 3: This breaks the problem into smaller pieces
        ‚Üí Each call handles one repetition, then calls itself with n-1

Step 4: Eventually n reaches 0, the base case triggers, and it all combines

Step 5: Test to ensure both methods give the same result!

PART 4: THE COMPLETE SOLUTION WITH COMMENTS

METHOD 1: LOOP APPROACH (Best for beginners)
---------------------------------------------

function repeat(string, n) {
  // Start with an empty result
  let result = "";

  // Loop n times
  for (let i = 0; i < n; i++) {
    // Add the string to our result each time
    result = result + string;
  }

  // Give back the final result
  return result;
}

Example Usage:
--------------
repeat("Ha", 3)     ‚Üí  "HaHaHa"
repeat("‚≠ê", 5)    ‚Üí  "‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê"
repeat("Hi", 0)     ‚Üí  ""  (nothing, because we don't loop at all)
repeat("*", 8)      ‚Üí  "********"
repeat("abc", 2)    ‚Üí  "abcabc"


METHOD 2: RECURSION APPROACH (For advanced learners)
-----------------------------------------------------

function repeat(string, n) {
  // BASE CASE: When to stop recursion
  if (n <= 0) {
    return "";  // Stop here and return empty string
  }

  // RECURSIVE CASE: Call ourselves with n-1
  return string + repeat(string, n - 1);
}

Example Usage (Same Results):
-----------------------------
repeat("Ha", 3)     ‚Üí  "HaHaHa"
repeat("‚≠ê", 5)    ‚Üí  "‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê"
repeat("Hi", 0)     ‚Üí  ""
repeat("*", 8)      ‚Üí  "********"
repeat("abc", 2)    ‚Üí  "abcabc"


BONUS: USING A WHILE LOOP (Another approach)
---------------------------------------------

function repeat(string, n) {
  let result = "";
  let count = 0;

  // Keep looping while count is less than n
  while (count < n) {
    result = result + string;
    count = count + 1;  // or: count++
  }

  return result;
}

PART 5: UNDERSTANDING THE LOGIC DEEPER
================================================================================

Let's trace through: repeat("Hi", 3)

LOOP METHOD STEP-BY-STEP:
-------------------------
Start:    result = ""
          string = "Hi"
          n = 3

Loop 1 (i=0):
  result = "" + "Hi"  ‚Üí  result = "Hi"

Loop 2 (i=1):
  result = "Hi" + "Hi"  ‚Üí  result = "HiHi"

Loop 3 (i=2):
  result = "HiHi" + "Hi"  ‚Üí  result = "HiHiHi"

Loop ends (i=3, but 3 < 3 is false, so we stop)

Return: "HiHiHi"  ‚úì


RECURSION METHOD STEP-BY-STEP:
------------------------------
repeat("Hi", 3)
= "Hi" + repeat("Hi", 2)
= "Hi" + ("Hi" + repeat("Hi", 1))
= "Hi" + ("Hi" + ("Hi" + repeat("Hi", 0)))
= "Hi" + ("Hi" + ("Hi" + ""))    ‚Üê BASE CASE: n=0 returns ""
= "Hi" + ("Hi" + "Hi")
= "Hi" + "HiHi"
= "HiHiHi"  ‚úì


WHY EACH STEP IS NECESSARY:
---------------------------
1. We start with an empty string because we're BUILDING the result from nothing
2. We loop exactly n times to add the string exactly n times
3. Each loop adds one copy of the string
4. We return the complete built string at the end

PART 8: COMMON MISTAKES TO AVOID
================================================================================

‚ùå Mistake 1: Forgetting to Initialize Your Result Variable
   Problem: You might write:
            for (let i = 0; i < n; i++) {
              result = result + string;  // But result doesn't exist yet!
            }
   
   Solution: Always start with: let result = "";
            before using it in the loop!

‚ùå Mistake 2: Using the Loop Variable Wrong
   Problem: Confusing what i means in the loop
            for (let i = 0; i < n; i++) {
              result = result + i;  // This adds 0, 1, 2... not the string!
            }
   
   Solution: Remember: i is just a counter. Use your string parameter:
            result = result + string;

‚ùå Mistake 3: Loop Condition Off By One
   Problem: Not understanding when the loop stops
            for (let i = 1; i < n; i++)  // Starts at 1 instead of 0
            // This runs n-1 times, not n times!
   
   Solution: Standard loop starts at 0 and goes while i < n:
            for (let i = 0; i < n; i++)
            // This runs exactly n times!

‚ùå Mistake 4: Forgetting the Return Statement
   Problem: Your function does the work but doesn't give back the result
            function repeat(string, n) {
              let result = "";
              for (let i = 0; i < n; i++) {
                result = result + string;
              }
              // Oops, forgot: return result;
            }
   
   Solution: Always return your final result:
            return result;

‚ùå Mistake 5: Not Handling Edge Cases (Like Negative Numbers)
   Problem: What happens if someone calls repeat("Hi", -5)?
            Your loop won't run at all (which is actually okay!)
            But it's good to be aware of this behavior.
   
   Solution: Test with edge cases:
            - repeat("test", 0)   ‚Üí Should be ""
            - repeat("test", -1)  ‚Üí Should be ""
            - repeat("test", 1)   ‚Üí Should be "test"

‚ùå Mistake 6: Modifying Parameters Inside the Function
   Problem: If you change n or string, you might get confused
   Solution: Don't modify your parameters; create new variables instead

PART 11: COMPLETE CORRECT SOLUTION
================================================================================

Below is the final, clean version of the solution that you should submit.
This is the RECOMMENDED approach for beginners:

```javascript
function repeat(string, n) {
  let result = "";
  for (let i = 0; i < n; i++) {
    result = result + string;
  }
  return result;
}
```

ALTERNATIVE: If you want to use recursion, here's the elegant version:

```javascript
function repeat(string, n) {
  if (n <= 0) {
    return "";
  }
  return string + repeat(string, n - 1);
}
```

ALTERNATIVE: Using a while loop:

```javascript
function repeat(string, n) {
  let result = "";
  let i = 0;
  while (i < n) {
    result = result + string;
    i++;
  }
  return result;
}
```

‚úì Recursion-based approach: Elegant, teaches important concepts, works perfectly
‚úì While-loop approach: Similar to for-loop but different style
‚úì All handle edge cases (0, 1, negative numbers)
‚úì All use only the parameters provided, no built-in repeat() method
‚úì All return strings with the correct length

All three approaches are CORRECT. Choose whichever makes most sense to you!

Troubleshooting:
----------------
Problem: "repeat is not defined"
Solution: Make sure you pasted your function code into the console FIRST,
          before trying to call it. The function must be defined before you
          use it.

Problem: I see "HaHaHaHa" but expected "HaHaHa"
Solution: Your loop is running one extra time. Check your loop condition:
          Use: for (let i = 0; i < n; i++)
          NOT: for (let i = 0; i <= n; i++)  // <= is wrong!

Problem: Nothing happens when I type repeat("Hi", 3)
Solution: Make sure you pressed Enter after typing the command, and check
          that you're in the Console tab (not the Elements tab).

Problem: My recursion version causes the page to freeze

          beginning of your function:
          if (n <= 0) return "";

          actually contains the code (didn't copy empty content).
