PART 2: KEY CONCEPTS YOU NEED TO UNDERSTAND

Concept 1: String Methods - The Tools in Your Toolkit
------------------------------------------------------
A String Method is like a tool in a toolbox. It's a pre-built function that 
JavaScript gives you to manipulate strings.

Think of strings like LEGO blocks. JavaScript provides special tools to:
- Split them apart (.split())
- Make them bigger (.toUpperCase())
- Make them smaller (.toLowerCase())
- Change the first letter (.charAt())

You don't have to build these tools yourself - they already exist! You just 
need to know how to use them.

Examples:
  "hello".toUpperCase() ‚Üí "HELLO"
  "HELLO".toLowerCase() ‚Üí "hello"
  "hello world".split(" ") ‚Üí ["hello", "world"]


Concept 2: Array Methods - Working with Collections
----------------------------------------------------
An Array Method is a tool for working with arrays (lists of items).

If strings are individual LEGO blocks, arrays are collections of blocks.
The most important array method you'll use is .join() - it combines items.

Think of it like this:
- .split(" ") breaks things apart (opposite of join)
- .join(" ") puts things together (opposite of split)

These are like opposite actions:
  "hello world".split(" ") ‚Üí ["hello", "world"]
  ["hello", "world"].join(" ") ‚Üí "hello world"

Notice: split and join are OPPOSITES. They undo each other!


Concept 3: String Indexing and Characters
------------------------------------------
Strings are made of characters, and you can access each character by position.

Think of a string like a row of boxes:
  "hello"
   01234    ‚Üê These are the positions (indices)

Position 0 = 'h'
Position 1 = 'e'
Position 2 = 'l'
Position 3 = 'l'
Position 4 = 'o'

You can get a character at a position using:
  "hello"[0] ‚Üí "h"
  "hello"[1] ‚Üí "e"

And you can get the rest of the string AFTER position 0 using:
  "hello".slice(1) ‚Üí "ello"
  "hello".substring(1) ‚Üí "ello"

This is how we capitalize: take the first character (uppercase it) and add 
the rest of the string (in lowercase).


Concept 4: Method Chaining - Doing Multiple Things at Once
-----------------------------------------------------------
Methods can be chained together, meaning you call one method, then call 
another method on the result!

Think of it like an assembly line:
- Raw string ‚Üí split() ‚Üí array of words ‚Üí join() ‚Üí finished sentence

Example:
  "  hello  world  ".trim().split(" ").join("-")
  
This does: trim spaces ‚Üí split by space ‚Üí join with dash

Each method works on the RESULT of the previous method!

PART 4: THE COMPLETE SOLUTION WITH COMMENTS

Below is the complete solution with detailed comments:

```javascript
// Function 1: words()
// Takes a sentence and returns an array of words
// INPUT: "hello world" ‚Üí OUTPUT: ["hello", "world"]
function words(str) {
  // .split(" ") breaks the string at every space
  // It creates an array with each word as an item
  return str.split(" ");
}


// Function 2: sentence()
// Takes an array of words and combines them into a sentence
// INPUT: ["hello", "world"] ‚Üí OUTPUT: "hello world"
function sentence(arr) {
  // .join(" ") takes every item in the array and combines them
  // The " " (space) goes between each item
  return arr.join(" ");
}


// Function 3: yell()
// Takes a string and makes it LOUD (all uppercase)
// INPUT: "hello" ‚Üí OUTPUT: "HELLO"
function yell(str) {
  // .toUpperCase() converts every letter to uppercase
  // Numbers and spaces stay the same
  return str.toUpperCase();
}


// Function 4: whisper()
// Takes a string and makes it quiet with asterisks
// INPUT: "HELLO" ‚Üí OUTPUT: "*hello*"
function whisper(str) {
  // .toLowerCase() converts every letter to lowercase
  // Then we add "*" at the beginning and end
  // We use + to combine strings (concatenation)
  return "*" + str.toLowerCase() + "*";
}


// Function 5: capitalize()
// Takes a string and fixes capitalization
// First letter uppercase, rest lowercase
// INPUT: "hELLO wORLD" ‚Üí OUTPUT: "Hello world"
function capitalize(str) {
  // str[0] gets the first character
  // .toUpperCase() makes it big
  // str.slice(1) gets everything AFTER the first character
  // .toLowerCase() makes the rest small
  // We combine them with +
  return str[0].toUpperCase() + str.slice(1).toLowerCase();
}
```

PART 8: COMMON MISTAKES TO AVOID
================================================================================

‚ùå Mistake 1: Forgetting to return the result
   Problem: function words(str) { str.split(" "); }
   This function does the work but returns "undefined"!
   Solution: Always use RETURN when you want to give back a value
   Correct: function words(str) { return str.split(" "); }

‚ùå Mistake 2: Using the wrong separator in split() or join()
   Problem: "hello-world".split(" ") ‚Üí ["hello-world"]
   The dash doesn't split anything because you split by SPACE, not dash!
   Solution: Match the separator to what you actually want
   Correct: "hello-world".split("-") ‚Üí ["hello", "world"]

‚ùå Mistake 3: Forgetting the asterisks in whisper()
   Problem: function whisper(str) { return str.toLowerCase(); }
   This returns lowercase but without the asterisks!
   Solution: Remember to add "*" at the beginning AND end
   Correct: return "*" + str.toLowerCase() + "*";

‚ùå Mistake 4: Using wrong method names
   Problem: "hello".uppercase() or "hello".UPPERCASE()
   JavaScript is CASE-SENSITIVE. It's toUpperCase, not uppercase
   Solution: Use exact method names (they're lowercase method names)
   Correct: "hello".toUpperCase() [note the camelCase]

‚ùå Mistake 5: Confusing slice() with split()
   Problem: str.split(1) to get the first character
   split() is for breaking strings, not getting characters!
   Solution: Use slice() or [0] for getting parts of strings
   Correct: str.slice(1) or str[0]

‚ùå Mistake 6: Forgetting edge cases (single letter strings)
   Problem: capitalize("") crashes or acts weird
   Solution: Test your functions with edge cases like "", "a", "A"
   Note: capitalize("") with our code would fail - discuss why!
