Concept 1: String Manipulation and Iteration
---------------------------------------------
Strings in JavaScript are sequences of characters. To split a string, you need
to:
1. Go through each character one by one (iteration)
2. Check if you've found the separator
3. Collect characters between separators into groups

ANALOGY: Imagine a sentence written on a long strip of paper. You want to cut
it into pieces wherever you see a dash (-). You go through character by
character, and whenever you find a dash, you cut the paper and start a new
piece.

Example with "a-b-c":
  Position: 0 1 2 3 4 5
  String:   a - b - c
            ^   ^   ^
  When we hit positions 1 and 3 (the dashes), we know to "cut" and start
  collecting a new piece.

Concept 2: Loops and Character-by-Character Processing
-------------------------------------------------------
You'll use a loop (typically for loop) to examine each character in the string.
In JavaScript, you can:
- Access individual characters using string[index]
- Check the length of a string using string.length
- Iterate from 0 to string.length - 1

Example:
  let str = "hello";
  for (let i = 0; i < str.length; i++) {
    console.log(str[i]);  // Prints each character
  }
  // Output: h e l l o (one per line)

Concept 3: Building Arrays Dynamically
---------------------------------------
To create the result array for split(), you can't know in advance how many
pieces you'll have. So you:
1. Create an empty array
2. As you process the string, push() new pieces into it
3. Return the final array when done

The push() method adds an element to the end of an array:
  let result = [];
  result.push("hello");
  result.push("world");
  console.log(result);  // ["hello", "world"]

Concept 4: String Concatenation and Building Pieces
----------------------------------------------------
As you loop through characters, you need to accumulate them into pieces:
1. Start with an empty string (currentPiece = "")
2. Add each character to it until you hit a separator
3. When you hit a separator, push the piece and reset

Example tracking:
  String: "a-b"
  Separator: "-"
  
  i=0: currentPiece = "a"
  i=1: Hit separator! Push "a", reset currentPiece = ""
  i=2: currentPiece = "b"
  End of string: Push "b"
  
  Result: ["a", "b"]

Concept 5: Joining Arrays with a Separator
-------------------------------------------
The opposite of split. You have an array and want to combine it into a single
string with a separator between elements:
1. Start with an empty result string
2. Loop through each element in the array
3. Add the element + separator to the result (except maybe the last one)
4. Return the final string

Example:
  Array: ["hello", "world"]
  Separator: "-"
  
  Start: result = ""
  Add "hello" ‚Üí result = "hello"
  Add "-" ‚Üí result = "hello-"
  Add "world" ‚Üí result = "hello-world"
  
  Return: "hello-world"

PART 3: STEP-BY-STEP ALGORITHM (Your Thinking Process)
================================================================================

FOR THE split() FUNCTION:

Step 1: Check if the separator exists in the string
      If not, return an array with just the whole string

Step 2: Create an empty array to hold your results

Step 3: Create a variable to accumulate the current piece you're building

Step 4: Loop through each character in the string
        - If the current characters match the separator:
          * Push the accumulated piece to the result array
          * Reset the current piece
          * Skip past all the characters in the separator
        - Otherwise:
          * Add the character to the current piece

Step 5: After the loop ends, don't forget the last piece!
        Push it to the result array

Step 6: Return the result array

FOR THE join() FUNCTION:

Step 1: Create an empty string to build your result

Step 2: Loop through each element in the array

Step 3: For each element:
        - Add it to the result string
        - If it's not the last element, add the separator after it

Step 4: Return the final result string

```javascript
// FUNCTION 1: split(string, separator)
// This function breaks a string into pieces based on a separator
function split(str, sep) {
  // Create an empty array to store the pieces we'll find
  const result = [];
  
  // If the separator is an empty string or doesn't exist in the string,
  // we need to handle it specially
  if (sep === "") {
    // Empty separator means split into individual characters
    for (let i = 0; i < str.length; i++) {
      result.push(str[i]);
    }
    return result;
  }
  
  // Start building the current piece
  let currentPiece = "";
  
  // Loop through each character position in the string
  let i = 0;
  while (i < str.length) {
    // Check if the next characters match our separator
    // We use substring to get a chunk of the string starting at position i
    if (str.substring(i, i + sep.length) === sep) {
      // We found the separator!
      // Add the piece we've been building to our results
      result.push(currentPiece);
      
      // Reset the current piece
      currentPiece = "";
      
      // Skip past the separator characters
      // If separator is 3 characters long, skip 3 positions
      i += sep.length;
    } else {
      // This character is NOT the start of a separator
      // Add it to the piece we're building
      currentPiece += str[i];
      
      // Move to the next character
      i++;
    }
  }
  
  // Don't forget the last piece! (There's no separator after it)
  result.push(currentPiece);
  
  return result;
}


// FUNCTION 2: join(array, separator)
// This function combines array elements into a string with a separator between them
function join(arr, sep) {
  // Start with an empty result string
  let result = "";
  
  // Loop through each element in the array
  for (let i = 0; i < arr.length; i++) {
    // Add the current element to our result
    result += arr[i];
    
    // Add the separator AFTER each element, but NOT after the last one
    // Check if this isn't the last element
    if (i < arr.length - 1) {
      result += sep;
    }
  }
  
  return result;
}
```

  arr = ["hi", "bye"]

  i = 2

Why This Works:
---------------
The split() function works because:
- We check each position to see if a separator starts there
- We accumulate characters until we find a separator
- We handle the final piece after the loop (the most common mistake!)

The join() function works because:
- We add each element sequentially
- We add separators between elements (but not after the last one)
- Checking (i < arr.length - 1) prevents an extra separator at the end

PART 8: COMMON MISTAKES TO AVOID
================================================================================

‚ùå Mistake 1: Forgetting the last piece in split()
   Problem: Your split() function returns one fewer element than expected
            Example: split("a-b", "-") returns ["a"] instead of ["a", "b"]
   Solution: Make sure you push the last currentPiece AFTER the loop ends
            Even if there's no separator at the end, the last piece needs
            to be added to the result array

‚ùå Mistake 2: Adding an extra separator at the end in join()
   Problem: Your join() function adds the separator after EVERY element
            Example: join(["a", "b"], "-") returns "a-b-" instead of "a-b"
   Solution: Check (i < arr.length - 1) before adding the separator
            This ensures you only add it BETWEEN elements, not after the last

‚ùå Mistake 3: Not handling empty separators in split()
   Problem: split("hello", "") doesn't return ["h", "e", "l", "l", "o"]
   Solution: Add a special case at the beginning to check if sep === ""
            If it is, just split into individual characters

‚ùå Mistake 4: Using incorrect string comparison in split()
   Problem: You compare str[i] === sep when sep might be multiple characters
   Solution: Use str.substring(i, i + sep.length) === sep instead
            substring() extracts a portion of the string to compare

‚ùå Mistake 5: Accidentally using the built-in split() or join()
   Problem: You call "hello".split("-") instead of split("hello", "-")
   Solution: Remember: the string/array comes FIRST in your functions
            Your function signatures are different from the built-in ones

‚ùå Mistake 6: Infinite loops with while loop
   Problem: Your while loop never increments i, causing it to run forever
   Solution: Make sure you increment i in BOTH branches of your if statement
            When you find a separator: i += sep.length
            When you don't: i += 1

‚ùå Mistake 7: Not initializing variables correctly
   Problem: Variables are undefined or have wrong starting values
   Solution: Initialize currentPiece = "", result = [], and i = 0 clearly
            Always start with empty arrays and strings, not undefined

```javascript
function split(str, sep) {
  const result = [];
  
  if (sep === "") {
    for (let i = 0; i < str.length; i++) {
      result.push(str[i]);
    }
    return result;
  }
  
  let currentPiece = "";
  let i = 0;
  
  while (i < str.length) {
    if (str.substring(i, i + sep.length) === sep) {
      result.push(currentPiece);
      currentPiece = "";
      i += sep.length;
    } else {
      currentPiece += str[i];
      i++;
    }
  }
  
  result.push(currentPiece);
  return result;
}

function join(arr, sep) {
  let result = "";
  
  for (let i = 0; i < arr.length; i++) {
    result += arr[i];
    if (i < arr.length - 1) {
      result += sep;
    }
  }
  
  return result;
}
```

Complete Browser Console Test Script:
-------------------------------------
Copy everything below into your browser console and run it at once:

```javascript
// Define the functions
function split(str, sep) {
  const result = [];
  
  if (sep === "") {
    for (let i = 0; i < str.length; i++) {
      result.push(str[i]);
    }
    return result;
  }
  
  let currentPiece = "";
  let i = 0;
  
  while (i < str.length) {
    if (str.substring(i, i + sep.length) === sep) {
      result.push(currentPiece);
      currentPiece = "";
      i += sep.length;
    } else {
      currentPiece += str[i];
      i++;
    }
  }
  
  result.push(currentPiece);
  return result;
}

function join(arr, sep) {
  let result = "";
  
  for (let i = 0; i < arr.length; i++) {
    result += arr[i];
    if (i < arr.length - 1) {
      result += sep;
    }
  }
  
  return result;
}

// Run tests and show results
console.log("=== SPLIT TESTS ===");
console.log("split('hello-world', '-'):", split("hello-world", "-"));
console.log("split('a,b,c', ','):", split("a,b,c", ","));
console.log("split('JavaScript', ''):", split("JavaScript", ""));
console.log("split('noSeparator', '-'):", split("noSeparator", "-"));

console.log("\n=== JOIN TESTS ===");
console.log("join(['hello', 'world'], '-'):", join(["hello", "world"], "-"));
console.log("join(['one', 'two', 'three'], ', '):", join(["one", "two", "three"], ", "));
console.log("join(['solo'], '-'):", join(["solo"], "-"));
```

Expected Console Output:

```

split('a,b,c', ','): Array [ "a", "b", "c" ]

```

Troubleshooting:
----------------
Problem: "split is not defined" in the console
Solution: Make sure you've pasted and executed the function definition
         in the console. Copy the entire function and paste it, then press Enter.

Problem: The function runs but returns unexpected results
Solution: Check your logic carefully against Part 5 (Understanding the Logic)
         Trace through one example step-by-step on paper
         Common issues: forgetting the last piece, extra separators, etc.

Problem: The HTML test file won't open
Solution: Make sure the file has the .html extension (not .txt)
         Right-click and choose "Open with" ‚Üí "Choose another app" ‚Üí Browser
         Or drag the file into your browser window

Problem: My split() function doesn't handle multi-character separators
Solution: Use str.substring(i, i + sep.length) instead of just str[i]
         This extracts multiple characters for comparison
