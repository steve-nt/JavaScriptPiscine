Concept 1: Array Indexing and Positions
----------------------------------------
In JavaScript, arrays are like shelves with numbered positions. The FIRST item
is at position 0, the SECOND at position 1, and so on.

Array:     [10,  20,  30,  40,  50]
Index:      0   1   2   3   4

When we reverse, we want to swap:
- Index 0 with Index 4
- Index 1 with Index 3
- Index 2 stays in the middle (it's the center)

This is like rearranging books on a shelf: take the first and last, swap them,
then move inward!

Concept 2: The Swap Operation (Exchanging Values)
-------------------------------------------------
To reverse properly, we need to SWAP elements. Swapping means exchanging the
positions of two values. Here's the trick:

‚ùå WRONG way:
  array[0] = array[4];    // Now you've overwritten index 0!
  array[4] = array[0];    // But array[0] is already gone!

‚úì RIGHT way (using a temporary container):
  let temp = array[0];     // Save the first value in a backup
  array[0] = array[4];     // Move the last value to the first position
  array[4] = temp;         // Move the saved value to the last position

Think of it like swapping glasses with a friend:
1. You hand your glasses to a helper (temp variable)
2. You take your friend's glasses and wear them
3. Your friend takes the saved glasses from the helper

Concept 3: Loop Control with Two Pointers
------------------------------------------
Instead of looping through ALL elements, we only need to go HALFWAY through
the array. We use two variables (pointers) that start from opposite ends and
move toward each other:

Array:  [1, 2, 3, 4, 5]
        ‚Üë           ‚Üë
     start         end

After first swap: [5, 2, 3, 4, 1]
                     ‚Üë       ‚Üë
                    start   end

After second swap: [5, 4, 3, 2, 1]
                        ‚Üë   ‚Üë
                       start end (they meet, we're done!)

Why only halfway? Because if you swap all the way, you'd undo your swaps!

Concept 4: Type Checking (Arrays vs. Strings)
----------------------------------------------
Arrays and strings are different:
- Arrays: Can be modified directly (mutable). You use reverse([1,2,3])
- Strings: Cannot be modified. You must build a new string and return it

We need to check WHAT type we received. In JavaScript:
  typeof [1,2,3] === 'object'  // Arrays are objects
  typeof "hello" === 'string'   // Strings are strings
  Array.isArray([1,2,3] === true // Better way to check for arrays

Concept 5: String Conversion to Array
--------------------------------------
Strings and arrays are similar but different. To reverse a string easily:

1. Split the string into individual characters: "abc".split('') ‚Üí ['a','b','c']
2. Reverse the array
3. Join them back together: ['c','b','a'].join('') ‚Üí "cba"

The split() method breaks a string into pieces, and join() puts them together.

PART 3: STEP-BY-STEP ALGORITHM (Your Thinking Process)
================================================================================

Step 1: Check if the input is an array or string
   - Use Array.isArray() to determine what we're working with
   - This tells us if we need to modify in-place (array) or return new (string)

Step 2: If it's a string, convert it to an array
   - Use the split('') method to break it into characters
   - This makes it easier to reverse

Step 3: Set up two pointers
   - One pointer at the start (index 0)
   - One pointer at the end (index length - 1)

Step 4: Loop while start pointer is less than end pointer
   - Swap the elements at these positions using a temporary variable
   - Move the start pointer forward (increment it)
   - Move the end pointer backward (decrement it)

Step 5: If the original input was a string, join the array back
   - Use the join('') method to combine characters back into a string
   - Return this new string

Step 6: If the original input was an array, it's already reversed!
   - Return the modified array (the mutation already happened)

// Main reverse function that works with both arrays and strings
function reverse(input) {
  // Check if the input is an array
  if (Array.isArray(input)) {
    // For arrays, we'll reverse in-place (modify the original)
    
    // Set up our two pointers: one at start, one at end
    let start = 0;                    // First position
    let end = input.length - 1;       // Last position
    
    // Keep swapping until we reach the middle
    while (start < end) {
      // Store the first element in a temporary variable (our backup)
      let temp = input[start];
      
      // Move the last element to the first position
      input[start] = input[end];
      
      // Put our saved element at the last position
      input[end] = temp;
      
      // Move the start pointer closer to the middle
      start++;
      
      // Move the end pointer closer to the middle
      end--;
    }
    
    // Return the modified array (it's been reversed in-place!)
    return input;
    
  } else if (typeof input === 'string') {
    // For strings, we can't modify them directly, so we create a new one
    
    // Split the string into an array of characters
    // "hello" becomes ['h', 'e', 'l', 'l', 'o']
    let chars = input.split('');
    
    // Set up our two pointers again
    let start = 0;
    let end = chars.length - 1;
    
    // Swap characters until we reach the middle
    while (start < end) {
      let temp = chars[start];
      chars[start] = chars[end];
      chars[end] = temp;
      
      start++;
      end--;
    }
    
    // Join the characters back into a string and return it
    // ['o', 'l', 'l', 'e', 'h'] becomes "olleh"
    return chars.join('');
  }
}

PART 5: UNDERSTANDING THE LOGIC DEEPER
================================================================================

Let's trace through a real example step-by-step:

Reversing the array [1, 2, 3]:

START: [1, 2, 3]
       ‚Üë     ‚Üë
    start  end

Iteration 1:
- temp = 1 (save the first element)
- array[0] = 3 (put 3 at the beginning)
- array[2] = 1 (put saved 1 at the end)
- Array is now: [3, 2, 1]
- start = 1, end = 1
- Loop condition: 1 < 1? NO! Loop stops.

RESULT: [3, 2, 1] ‚úì Correct!

Why do we stop when start >= end?
Because that's when we've reached the middle. For odd-length arrays, the middle
element doesn't need swapping. For even-length arrays, when start equals end,
all elements have been swapped.

Example with array [1, 2, 3, 4]:

START: [1, 2, 3, 4]
       ‚Üë        ‚Üë
    start      end

Iteration 1: Swap 1 and 4 ‚Üí [4, 2, 3, 1]
             start = 1, end = 2

Iteration 2: Swap 2 and 3 ‚Üí [4, 3, 2, 1]
             start = 2, end = 1
             
Loop condition: 2 < 1? NO! Loop stops.

RESULT: [4, 3, 2, 1] ‚úì Correct!

Why This Approach is Efficient:
‚úì Time Complexity: O(n/2) which simplifies to O(n) - we visit each element
‚úì Space Complexity: O(1) - we only use one extra variable (temp), not extra space

PART 8: COMMON MISTAKES TO AVOID
================================================================================

‚ùå Mistake 1: Forgetting to declare the temp variable
   Problem: You swap input[start] with input[end] without saving one first,
            causing data loss. Example: array[0] = array[4]; array[4] = array[0]
            Both end up with the same value!
   Solution: Always use: let temp = array[start]; before overwriting

‚ùå Mistake 2: Using Array.reverse() accidentally
   Problem: The assignment explicitly forbids using Array.reverse(). If you
            use it, your solution won't pass the tests!
   Solution: Implement the swap logic yourself. Never call .reverse() on the input.

‚ùå Mistake 3: Not handling both arrays AND strings
   Problem: You write code that only handles arrays and crash when given a string
   Solution: Use if/else or if/else if to check the type. Use Array.isArray()
            and typeof to determine what you're working with.

‚ùå Mistake 4: Reversing the entire array instead of using two pointers
   Problem: You loop through the whole array and cause unnecessary operations
            or even undo your own swaps. This is inefficient!
   Solution: Use two pointers moving toward the center. Stop when start >= end.

‚ùå Mistake 5: Forgetting to return the result
   Problem: You modify the array correctly but forget to return it (for arrays)
            or forget to return the joined string (for strings)
   Solution: Always include return statements. For arrays: return input;
            For strings: return chars.join('');

‚ùå Mistake 6: Modifying strings directly
   Problem: Strings are immutable in JavaScript. You can't do myString[0] = 'x'
   Solution: Convert to array (split), modify the array, then convert back (join)

```javascript
function reverse(input) {
  // Handle arrays: reverse in-place
  if (Array.isArray(input)) {
    let start = 0;
    let end = input.length - 1;
    
    while (start < end) {
      let temp = input[start];
      input[start] = input[end];
      input[end] = temp;
      start++;
      end--;
    }
    
    return input;
  }
  
  // Handle strings: create a new reversed string
  if (typeof input === 'string') {
    let chars = input.split('');
    let start = 0;
    let end = chars.length - 1;
    
    while (start < end) {
      let temp = chars[start];
      chars[start] = chars[end];
      chars[end] = temp;
      start++;
      end--;
    }
    
    return chars.join('');
  }
}
```

Problem: "Array.prototype.reverse is undefined" error
Solution: This is expected! The assignment intentionally disables it so you
          can't accidentally use it. Just ignore this error message.

Problem: My test cases are showing wrong values
Solution: Check that you:
          1. Have the temp variable for swapping
          2. Are using start < end (not <=)
          3. Returning the result
          4. Using split() and join() for strings

Problem: Array isn't reversing
Solution: Make sure you're swapping correctly:
          let temp = array[start];
          array[start] = array[end];
          array[end] = temp;
          If any step is missing, the swap won't work!

Problem: String isn't reversing
Solution: Remember strings need split() at the start and join() at the end.
          Your code should look like: input.split('') ... .join('')
