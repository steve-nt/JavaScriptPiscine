- So a dog ages differently on each planet!

PART 2: KEY CONCEPTS YOU NEED TO UNDERSTAND
================================================================================

Concept 1: What is an Orbital Period?
--------------------------------------
An orbital period is how long it takes a planet to complete one full orbit 
around the sun. On Earth, that's 365.25 days. On Mercury (closest to the sun),
it only takes about 88 days. On Neptune (farthest), it takes about 60,000 days!

Think of it like this: If you're on a carousel horse, one full spin is your 
"year." The horse spins one way in 10 seconds. That's your orbital period!
If you ride a slower carousel (like Saturn), one spin might take 30 seconds.

Concept 2: Converting Seconds to Earth Years
----------------------------------------------
First, we need to convert seconds into something we understand: Earth years.

How many seconds are in ONE Earth year?
  - 1 year = 365.25 days (accounting for leap years)
  - 1 day = 24 hours
  - 1 hour = 3600 seconds
  - So: 365.25 √ó 24 √ó 3600 = 31,557,600 seconds

Formula: Age in Earth years = Age in seconds √∑ 31,557,600

Example: 1,000,000,000 seconds √∑ 31,557,600 = 31.688 Earth years

Concept 3: Converting Earth Years to Planet Years
---------------------------------------------------
Once we know the age in Earth years, we need to convert to the specific planet.

Why? Because a "year" is different on each planet!
  - On Mercury, it takes only 0.2408 Earth years to orbit the sun
  - On Neptune, it takes 164.79 Earth years to orbit the sun

If a dog is 31.688 Earth years old, how many years old is it on Mercury?
  Age on planet = Age in Earth years √∑ Orbital period of that planet
  Age on Mercury = 31.688 √∑ 0.2408 = 131.6 Mercury years

This makes sense: A "year" on Mercury is much shorter than an Earth year,
so the dog would be much older in Mercury years!

Concept 4: The 7 Dog Years Rule
--------------------------------
Humans and dogs age at different rates. The rule of thumb is:
  Dog years = Planet years √ó 7

This means 1 year for a human = 7 years for a dog (aging-wise).

So if a dog is 31.688 Earth years old:
  Dog years on Earth = 31.688 √ó 7 = 221.82 dog years

Concept 5: Rounding to 2 Decimal Places
----------------------------------------
Our answer needs to be rounded to 2 decimal places (2 digits after the decimal).

221.8236 rounds to 221.82
10.1456 rounds to 10.15
5.999 rounds to 6.00

JavaScript has a method: number.toFixed(2) that does this!

Step 1: Gather the orbital periods for each planet
  - Create an object that maps planet names to their orbital periods

Step 2: Get the orbital period for the planet the user asked about
  - Look up the orbital period from your object

Step 3: Convert the dog's age from seconds to Earth years
  - Divide the age in seconds by 31,557,600

Step 4: Convert from Earth years to years on that planet
  - Divide the Earth years by the orbital period

Step 5: Apply the dog years rule (multiply by 7)
  - Multiply the planet years by 7

Step 6: Round to 2 decimal places
  - Use .toFixed(2) method

Step 7: Return the result
  - Make sure you convert it back to a number

// Our function takes two parameters:
// 1. planetName - a string like "earth", "mars", etc.
// 2. ageInSeconds - a number representing the dog's age in seconds

function dogYears(planetName, ageInSeconds) {
  // Step 1: Create an object with orbital periods
  // Each planet is a key, and the orbital period (in Earth years) is the value
  const orbitalPeriods = {
    earth: 1.0,
    mercury: 0.2408467,
    venus: 0.61519726,
    mars: 1.8808158,
    jupiter: 11.862615,
    saturn: 29.447498,
    uranus: 84.016846,
    neptune: 164.79132
  };

  // Step 2: Get the orbital period for the requested planet
  // Convert the planet name to lowercase in case the user types "EARTH" or "Earth"
  const orbitalPeriod = orbitalPeriods[planetName.toLowerCase()];

  // Step 3: Convert seconds to Earth years
  // 31,557,600 is the number of seconds in one Earth year
  const secondsPerEarthYear = 31557600;
  const ageInEarthYears = ageInSeconds / secondsPerEarthYear;

  // Step 4: Convert Earth years to years on the target planet
  // A year on the planet is different from an Earth year!
  // For example, a year on Mercury is much shorter (0.2408 Earth years)
  // So the age in that planet's years will be bigger
  const ageInPlanetYears = ageInEarthYears / orbitalPeriod;

  // Step 5: Apply the "7 dog years per human year" rule
  // This tells us how much faster dogs age compared to humans
  const ageInDogYears = ageInPlanetYears * 7;

  // Step 6: Round to 2 decimal places
  // .toFixed(2) returns a string, so we wrap it in Number() to convert back to a number
  const roundedAge = Number(ageInDogYears.toFixed(2));

  // Step 7: Return the final answer
  return roundedAge;
}

// Example 2: The same dog on different planets
console.log(dogYears("mercury", 1000000000));  // Output: 921.12
console.log(dogYears("mars", 1000000000));     // Output: 117.96
console.log(dogYears("jupiter", 1000000000));  // Output: 18.69

// Why are these different? Because a "year" is different on each planet!
// The closer to the sun, the shorter a year, so the dog ages faster.

Notice: The dog is older in dog years on Mercury because a Mercury year is 
much shorter than an Earth year. The same amount of time (31.688 Earth years) 
equals more Mercury years (131.60 years).

would equal 420 "dog minutes". It's the same amount of time, but in smaller 
units, the number is much bigger!

   Problem: dogYears("Earth", 1000000000) won't work if you don't convert to lowercase
   Solution: Always use planetName.toLowerCase()

   Problem: You get 221.81566... instead of 221.82
   Solution: Use .toFixed(2) and Number() conversion

‚ùå Mistake 3: Wrong order of operations
   Problem: You multiply by 7 before converting to planet years
   Solution: Follow the steps: seconds ‚Üí Earth years ‚Üí planet years ‚Üí dog years

‚ùå Mistake 4: Hard-coding the seconds value
   Problem: Creating a different function for each planet
   Solution: Use parameters and objects to make it flexible

   Problem: Your function works for Earth but breaks for Mercury
   Solution: Test with all planets!
